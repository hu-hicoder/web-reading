# Gitの使い方、GitHubの使い方
## GitとGitHubの違い
- Gitは「自分の書いたコードの履歴を管理するツール」、GitHubは「世界中の人に自分の書いたコードを共有できるWebサービス」
- GitHubはGitの上に成り立っている。逆に、GitはGit単体で使うことができる。
